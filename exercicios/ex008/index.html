Principais formatações

Negrito / Destaque

Nesta frase tempo um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em destaque com a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

E nesse outro parágrafo temos mais uma área marcada com a tag <mark>

A cor do texto marcado pode ser modificado através da CSS

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Lembrando que a tag <big> para deixar o texto grande está obsoleta e pode sumir a qualquer momento

A tag <small> serve para deixar o texto pequeno e acredita-se não ter caído em desuso por ser designada a textos legais

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase ou indicar que ele foi adicionado depois. Usando a tag <ins>

Existe também o texto sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x30-74. Usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O. Usando a tag <sub>
