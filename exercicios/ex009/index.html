Código fonte/ Pré Formatação

A tag <code> é usada para demarcar partes do texto que são códigos com fonte monoespaçada.

Já a tag <pre> (preformatted) é usada para fazer com que a formatação de texto feita no HTML seja mantida na viewport.

O comando document.GetElementById('teste') é escrito na linguagem JavaScript.

            
num = int(input('Digite um número:'))
    if num % 2 == 0:
        print(f'O número {num} é PAR.')
    else:
        print(f'O número {num} é ÍMPAR.')
    print('Fim do programa.')
